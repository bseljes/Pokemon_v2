Pokemon Red Inspired Learning Project - Milestone 3 Todo
=======================================================

Goal:
Create a simple backend prototype for tall grass and random encounters while keeping the battle itself
as a placeholder.

Milestone 3: Tall Grass and Random Encounters
---------------------------------------------

[ ] Review the current GrassTriggerController.cs.
[ ] Make the player track whether they are currently inside tall grass.
[ ] Make tall grass detection update the GameManager or a player state script.
[ ] Add an Encounter state to GameManager.
[ ] Prevent normal player interaction while in Encounter state.
[ ] Decide when encounter rolls happen, such as every few steps or every few seconds while moving.
[ ] Create a simple encounter chance value, such as 10%.
[ ] Roll for an encounter only while the player is in tall grass.
[ ] Roll for an encounter only while the player is moving.
[ ] Print encounter roll results to the Console for debugging.
[ ] Create simple PokemonData or CreatureData for test encounter information.
[ ] Add at least three test creatures.
[ ] Give each creature a name.
[ ] Give each creature a level or level range.
[ ] Give each creature a simple rarity or encounter weight.
[ ] Create an EncounterTable for the tall grass area.
[ ] Pick a creature from the EncounterTable when an encounter succeeds.
[ ] Show a text bubble or popup saying what creature appeared.
[ ] Switch GameManager to Encounter state when a creature appears.
[ ] Add a temporary key press to exit the encounter and return to Gameplay.
[ ] Make sure encounters do not trigger while dialogue or inventory is open.
[ ] Save the scene and test the full loop in Play Mode.
